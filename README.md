# CS 330 Computational Graphics and Visualization Portfolio

This repository contains my portfolio materials for CS 330: Computational Graphics and Visualization. The submitted artifacts include my final 3D scene project and my Design Decisions document.

## How do I approach designing software?

I start with the main goal and break it down into smaller steps. Through this project, I learned how to create an entire 3D scene using simple shapes. I learned how to position objects correctly, change their size and orientation, and add textures and lighting. I worked step by step: first, I created simple shapes, saw how they looked, then adjusted their size, color, and lighting. This approach is helpful because large tasks become easier when broken down into smaller parts.

## How do I approach developing programs?

I develop programs one function at a time. When creating the 3D scene, I carefully checked every step: how the objects move, whether the textures load correctly, and if the lighting works well. I constantly tested and improved the scene. At first, I just added shapes, but then I began to understand how different parts of the program work together. By the end, I had a good grasp of how the code relates to what the user sees.

## How can computer science help me in reaching my goals?

Computer science teaches you to solve problems that are useful everywhere. From this project, I learned how code can create images and visual effects. In the future, this knowledge will help me in my studies and work — especially in creating interfaces, data analysis, and web development. I realized that to achieve a good result, you need not only to code, but also to plan, test, and think creatively.
